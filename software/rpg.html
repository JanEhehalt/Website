Current state

Currently we are working on the combat system. We are trying to implement an tactical turn based system. You have to think about every turn of yours carefully and you can choose whether to attack, by sword or maybe by spell, or to move or maybe even run away... We still aren't sure how it should look in the end. But in general the game has many major features implemented. We can easily create a game world, where we can create NPCs you can talk with and e.G. signs which you can examine for information, e.G. to fulfull a quest you have where you have to find information inside the gameworld.

About

At the moment Jonathan and me are working together on a new Game. This Game will be an RPG in pixelart style. It will have an more or less deep storyline. You, as a player, have to make decisions which will be influencing the course of the story. On your way you have to fight enemies in a rather tactical turn based combat system. It is uncertain when this game will be released, because it takes a lot of time. The game is written in Java using the LibGDX engine.
